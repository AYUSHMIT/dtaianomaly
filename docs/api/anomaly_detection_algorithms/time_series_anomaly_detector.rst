Time Series Anomaly Detector
============================

This class forms the base of all anomaly detectors in ``dtaianomaly``. It provides a
simple to use interface for anomaly detection through the :py:meth:`~dtaianomaly.anomaly_detection.TimeSeriesAnomalyDetector.fit`
for training and :py:meth:`~dtaianomaly.anomaly_detection.TimeSeriesAnomalyDetector.decision_function`
to effectively detect anomalies.
